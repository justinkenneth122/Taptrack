## 🚀 Quick Start

### For Admins (How to Use)
```
1. Go to Admin Panel → Attendance (📋 icon)
2. Select an Event (required)
3. Apply optional filters (Program, Year Level, Search)
4. View attendance records instantly
5. Export to CSV if needed
```

```
1. Upload 3 PHP files to pages/admin/

3. Create database indexes (optional but recommended)
4. Test using provided test guide
5. Go live!
```
